# Screenshots

This directory holds the screenshots for the main README.

## Required Screenshots

Please capture the following and save them here:

| File | What to Capture |
|------|-----------------|
| `01-dashboard.png` | Main dashboard with **at least 2–3 sessions** across **2 different projects**, and **one session selected** so the Session details pane (middle column) is visible with repository info. Bonus: if a session has a linked PR, capture that. |
| `02-create-session.png` | The **"New Session"** form fully visible, ideally with a project selected and the **"Create worktree" toggle set to On**. |
| `04-help-menu.png` | The **help menu** (`?` key) showing the keyboard shortcuts. |

## Tips for Good Screenshots

- Use a **dark terminal** (matches the app's default theme)
- **Terminal size**: around **100×30** or larger so all UI elements fit
- Make sure the **macOS window chrome is visible** (red/yellow/green buttons) if possible
- The dashboard looks best when sessions are grouped by project — create a few sessions first
- For the help menu, make sure it's centered and not cut off
